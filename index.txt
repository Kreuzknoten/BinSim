Wie man die Visual Studio Code für das Projekt konfiguruert

- Node.js installieren über Browser
- Powershell als Administrator ausfühern und Befehl eingeben    
    - Set-ExecutionPolicy -ExecutionPolicy RemoteSigned
- in Powershell Konsole von VSC eingeben
    - npm install -g typescript
    - npm install webpack webpack-cli ts-loader -D
    - npm install -g typescript -D
- das Projekt kann jetzt mit diesem Befehl gebaut werden
    - npm run build 

- "Live Server" über VSC erweizterungen installieren
    - "index.html" mit Live Server öffnen

- "Prettier" über VSC erweizterungen installieren
    - VSC Settings Menü
        - Format On Save = true
        - Default Formatter = Prettier
        - Print Width = 100
